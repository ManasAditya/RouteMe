# 🚀 RouteMe AI

**RouteMe AI** is an intelligent travel planning platform that helps users discover destinations, optimize routes, estimate travel expenses, and navigate efficiently using AI-powered recommendations and interactive maps.

🌐 **Live Demo:** https://route-me-six.vercel.app/

---

## ✨ Features

* 🤖 **AI Travel Assistant**

  * Get intelligent travel recommendations.
  * Ask travel-related questions in a chatbot interface.

* 🗺️ **Interactive Maps**

  * View destinations on an interactive map.
  * Get optimized routes between locations.

* 📍 **Route Optimization**

  * Calculate efficient travel paths.
  * Display distance and travel information.

* 💰 **Expense Tracker**

  * Track travel expenses.
  * Manage trip budgets.

* 📷 **AI Image Scanner**

  * Scan landmarks or locations using images.
  * AI identifies and provides information about the place.

* 🚗 **Transport Selection**

  * Compare different transport options.
  * Choose the most suitable travel method.

* 📱 **Responsive Design**

  * Works seamlessly across desktop, tablet, and mobile devices.

* 🎨 **Modern UI**

  * Smooth animations.
  * Glassmorphism design.
  * Dark theme interface.
